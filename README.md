# nife-vm-agent-release

Binary distribution repository for [nife-vm-agent](https://github.com/nifetency/nife-vm-agent) — the VM agent that runs on managed compute nodes as part of the [Nife](https://nife.io) platform.

This repo hosts pre-built release binaries only. For source code, issues, and contributions, see [nifetency/nife-vm-agent](https://github.com/nifetency/nife-vm-agent).

## Download

Grab the latest release from the [Releases page](https://github.com/nifetency/nife-vm-agent-release/releases/latest). Builds are available for:

- Linux (386, amd64)
- macOS (amd64)
- Windows (386, amd64)

## Installation

```bash
# Example for Linux amd64
curl -LO https://github.com/nifetency/nife-vm-agent-release/releases/latest/download/nife-vm-agent-linux-amd64.tar.gz
tar -xzf nife-vm-agent-linux-amd64.tar.gz
./nife-vm-agent --version
```
